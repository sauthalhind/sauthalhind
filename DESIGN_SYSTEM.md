# Design System

## Visual direction

- Clean Arabic newsroom
- RTL-first layout
- White surfaces with soft borders and subtle shadows
- Deep green and teal accents with gold for emphasis
- Large readable headlines

## Typography

- Arabic headlines should feel editorial and bold
- Supporting text should remain calm and readable on mobile
- Keep line heights generous for Arabic body copy

## Layout rules

- Mobile first
- Cards should stack vertically on small screens
- Hero story should always be the most prominent block
- Categories and latest news should be secondary, but still visible

## Components

- Rounded cards
- Pill buttons
- Simple bordered inputs
- Clear CTA hierarchy
- Image-driven news cards

## Brand usage

- Use the `sauthalhind.png` logo in header, footer, and admin branding
- Avoid over-rounding the logo crop
- Preserve Arabic and English label pairing where already used

## Interaction

- Buttons should feel responsive
- Loading states should be visible
- Admin actions should show immediate status feedback
