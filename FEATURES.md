# Features

## Public site

- Arabic homepage
- Live trending rail
- Latest news list
- Category chips and category page
- Article detail pages
- Search page
- English landing page
- Footer with newsletter and quick links

## Admin CMS

- Create story
- Edit story
- Delete story
- Publish / draft toggle
- Cover photo upload
- File upload
- Category selection
- Basic workflow controls
- Diagnostics panel

## Platform support

- Supabase-backed CRUD
- Upload endpoint
- SEO sitemap and robots
- Feed output
- Live update sync between admin and homepage

## Reliability

- Build passes in production mode
- Browser cache is used as fallback when needed
- Admin actions recover from request failures
